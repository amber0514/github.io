# 兒童學習遊戲樂園 🎮

一個專為兒童設計的網頁學習遊戲，包含三個互動遊戲和心得日記功能。

## 開啟方式

直接用瀏覽器開啟 `kids-learning-games.html` 即可遊玩，無需安裝任何軟體或網路連線（字型除外）。

---

## 遊戲一：記憶配對 🧠

**目標**：翻開卡片，找出所有相同的圖案配對。

**玩法**：
1. 點擊卡片翻開，記住圖案位置
2. 連續翻開兩張，圖案相同即配對成功
3. 全部 8 組配對完成即過關

**技巧**：記住翻過的卡片位置，下次直接翻對應的卡片。

---

## 遊戲二：算術挑戰 🔢

**目標**：在四个選項中選出正確的加減法答案。

**玩法**：
1. 畫面上顯示一道加減法題目（例如 5 + 3 = ?）
2. 從四個按鈕中選出正確答案
3. 答對得一分，自動跳下一題
4. 答錯該按鈕會停用，可繼續選擇

**題目範圍**：數字 1~10 的加減法，減法保證答案為正數。

---

## 遊戲三：拼字遊戲 🔤

**目標**：根據 emoji 圖案和提示，拼出正確的英文單字。

**玩法**：
1. 看到一個 emoji 圖案（如 🐱）和中文提示
2. 下方鍵盤點擊字母，拼出完整的英文單字
3. 拼對字母會填入空格，拼錯字母會變灰
4. 完整拼出單字即得一分，自動換下一題

**單字列表**：CAT、DOG、FISH、BIRD、COW、PIG、ELEPHANT、LION

---

## 心得日記 📝

記錄每次玩遊戲的學習心得。

**功能**：
- 填寫姓名
- 選擇玩的遊戲
- 記錄心情（開心 / 普通 / 有點難 / 超級興奮）
- 寫下學習心得
- 所有記錄會儲存在瀏覽器中，關掉網頁也不會消失

---

## 操作方式

| 方式 | 說明 |
|------|------|
| 點選首頁卡片 | 進入對應遊戲 |
| 點選導航按鈕 | 快速切換遊戲 / 首頁 |
| 點選 `← 返回首頁` | 回到遊戲選擇畫面 |
| 點選 `重新開始` | 重置當前遊戲分數 |

---

## 技術說明

- 純 HTML + CSS + JavaScript，單一檔案
- 使用 CSS 漸層、動畫與圓角設計，適合兒童操作
- 心得日記使用瀏覽器 localStorage 儲存
- 響應式設計，手機與電腦皆可使用
- 字型採用 Google Noto Sans TC（無網路時使用系統字型）
